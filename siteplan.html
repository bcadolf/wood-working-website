Website Plan

Website Name:

It Begins With Wood

The name represents that the focus of the website is all about wood based knowledge and projects.

Website Purpose:

The website will provide a few pages about woodworking as a hobby, some personal experiences and some general knowledge. This is for anyone who desires to know more about woodworking.

Scenarios:

  • What is woodworking?
  • How can I get started?
  • Why do you like woodworking?
  • What projects do you start with?

Color Scheme:

  1. Primary Color #3e2f1d
  2. Secondary Color #a6b17b
  3. Accent Color #5ba75b
  4. Background Color #d2b48c
  5. Text #060419

Typography:

Inter Sans-Serif
